# Music Swipe

A Tinder-style music discovery app that helps users find new songs through swipe-based feedback and create playlists across multiple platforms.

## Core Idea
- Swipe-based music discovery (like Tinder)
- Multi-level feedback (Hate / Like / Love)
- Personalized recommendations
- Playlist creation on music platforms (Spotify first)

## Planned Enhancements
- Multi-level swipe gestures (Left = Hate, Right = Love, Up/Down = Like)
- Improved personalization using weighted preferences
- Support for multiple music platforms

## Tech Stack (Planned)
- Frontend: React + TypeScript
- Backend: NestJS
- Database: PostgreSQL + Prisma
- Recommendation Engine: Python (future)
